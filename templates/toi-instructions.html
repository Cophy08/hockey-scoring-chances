Time on Ice Database - API docs

Input

You know how to use the form to search the TOI database, this is the API to search the database for your own apps. All responses are in JSON. To use the API simply go to /toi/api/<int:gameID>/<int:period>/<int:timeRemaining> and fill it in with the appopriate values.

gameID:
This is the unique NHL identifier of each game, it takes the form of: yyyy0ddddd where the first four y's are the year and the last 5 d's are the unique game id, i.e. 2012020123

period
The number of the period. i.e. 1st Period is 1, 3 Period is 3. OT's continue with this trend i.e. 1OT is 4, 2OT is 5 etc.

timeRemaining
In seconds this is the number of seconds remaining, this value is between 0 and 1200.

Example: /toi/api/2012020123/3/1023

Output

There are six values in the JSON string that will be returned:

error:
This will either be true or false. True if it the API was able to successfully return a value, otherwise it is false. The last four variables only return a value if error is true.

message
Specifies why the API was not (or was) able to return a value.

team1name
The name of the first team.

team2name:
The name of the second team.

team1
The roster of the first team on the ice at this time.

team2
The roster of the second team on the ice at this time.
